Privacy Policy for Grammar Corrector

Effective Date: May 30, 2026

1. Introduction

Grammar Corrector ("the Extension") is a browser utility built with user data safety as the core priority. This document outlines how data variables are handled on-device. This Extension is developed and operated by the independent software developer V Jayashankar, listing across digital stores under the official developer name 'Jay Bro' (the "Service Provider").

2. Zero Data Collection & Transit Safety

The Extension does not collect, monitor, store, or transmit any personal data, user account identifiers, active browsing histories, or the text you choose to highlight. All runtime operational logic is run strictly in-memory locally. Any basic background web platform handshakes use secure cryptographic communication protocols (HTTPS/TLS).

3. Local AI Processing Framework

The Extension utilizes Google Chrome's built-in on-device intelligence framework (Gemini Nano) to analyze and correct structural text layouts. Because this AI engine is fully integrated into your native browser environment, all text processing happens entirely on your local machine. Your text variables are never synchronized, sent to our servers, or shared with external third-party cloud servers.

4. Required Browser Permissions

To deliver local grammar corrections, the Extension requests standard browser access permissions, used strictly for local execution hooks:

  • Context Menus: Enables the system to display a "Correct Grammar" action option when you execute a right-click command on highlighted text components.
  • Active Tab & Scripting: Allows the system layout to identify the text characters highlighted by the user and seamlessly replace them with the AI-optimized variant locally.
  • Storage: Saves your custom options (such as preferred localized English dialects and editing modes) inside your local browser folder directory.

5. Third-Party Services & Donation Portal Disclosures

  • Ecosystem Rules: Because this utility interfaces with built-in Chrome systems, runtime performance parameters follow the overarching Google Privacy Standards.
  • Support Channels & System Isolation: The Extension user interface includes an optional "Support Me" link directing users to external platforms (such as PayPal or Ko-fi) for voluntary community tipping. The Extension code does not host payment processing modules, capture billing profiles, or run transactional cards.
  • Merchant Line Isolation: These optional creator contributions are managed completely off-site. They are entirely separate from, and do not utilize, the business billing infrastructure, payment gateway clearings, or commercial settlement networks assigned to the developer's Google Play Console merchant account profile.

6. Localized Data Erasure & Deletion Rights

Because the Extension runs on an anonymous, zero-retention client structure, there are no remote user databases, analytical accounts, or customer profiles containing data records to request for deletion. Users can instantly and completely purge all localized configurations at any moment simply by removing the extension using the standard native browser extension management control screen (chrome://extensions).

We may update this Privacy Policy over time to maintain complete compliance with changing Chrome Web Store Developer Program Policies. Updates go live instantly upon the adjustment of the effective date at the top of this page.
